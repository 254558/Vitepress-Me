# 索引
## 一、索引是什么
DataFrame 有两套索引：
- **行索引 index**：每一行的编号/标签，在表格最左边
- **列索引 columns**：每一列的名字

```python
import pandas as pd
data = {
    "姓名":["张三","李四","王五"],
    "年龄":[18,20,22]
}
df = pd.DataFrame(data)
print(df)
```
输出：
```
   姓名  年龄
0  张三  18
1  李四  20
2  王五  22
```
左边 `0 1 2` 就是**默认行索引**。


## 二、获取索引
### 1. 获取行索引
```python
df.index
```

### 2. 获取列索引（列名）
```python
df.columns
```

## 三、修改/设置行索引
### 1. 用现有列当索引
```python
df = df.set_index("姓名")
```
把「姓名」列变成行索引。

### 2. 重置索引（恢复默认 0,1,2...）
```python
df = df.reset_index()
# 原地修改
df.reset_index(inplace=True)
```

### 3. 自定义索引
```python
df.index = ["a","b","c"]
```

## 四、两大索引取值神器
### 1. `loc` → 按**标签索引**取值
按**行名、列名**取，能切片、能条件筛选
```python
# 取标签为0的行
df.loc[0]

# 取指定行+指定列
df.loc[0:1, ["姓名","年龄"]]
```

### 2. `iloc` → 按**数字位置索引**取值
按**第几行、第几列**，只认数字下标
```python
# 第0行、第1列
df.iloc[0, 1]

# 前2行、所有列
df.iloc[:2, :]
```

### 口诀
- **loc 看名字**（标签）
- **iloc 看位置**（下标）
