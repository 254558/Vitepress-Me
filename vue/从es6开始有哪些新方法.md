# ES6+ 常用新方法/语法 速查表
## ES6 (2015)
| 方法/语法 | 作用 |
|---------|------|
| `let` / `const` | 块级作用域变量，替代 `var`，解决变量提升、重复声明问题 |
| 箭头函数 `() => {}` | 简化函数写法，不绑定自身 `this`，继承外层作用域 |
| 解构赋值 | 快速从数组/对象中提取值 |
| 展开运算符 `...` | 数组/对象拷贝、合并、函数传参 |
| 模板字符串 `` ` `${}` `` | 方便拼接字符串、支持换行、嵌入变量 |
| 函数默认参数 | 给函数参数设置默认值 |
| `Promise` | 解决异步回调地狱，异步编程基础 |
| `Array.find()` | 查找数组中第一个满足条件的元素 |
| `Array.findIndex()` | 查找数组中第一个满足条件的元素索引 |

---

## ES7 (2016)
| 方法/语法 | 作用 |
|---------|------|
| `Array.includes()` | 判断数组是否包含指定值，返回布尔值 |
| 幂运算符 `**` | 计算乘方，替代 `Math.pow()` |

---

## ES8 (2017)
| 方法/语法 | 作用 |
|---------|------|
| `async / await` | 异步编程终极方案，让异步代码写法像同步代码 |
| `Object.values()` | 获取对象所有值，组成数组 |
| `Object.entries()` | 获取对象所有键值对，组成二维数组 |
| `String.padStart()` | 字符串头部补全指定长度 |
| `String.padEnd()` | 字符串尾部补全指定长度 |

---

## ES9 (2018)
| 方法/语法 | 作用 |
|---------|------|
| 对象展开运算符 `...` | 对象拷贝、合并、修改属性 |
| `Promise.finally()` | Promise 无论成功/失败，最终都会执行 |

---

## ES10 (2019)
| 方法/语法 | 作用 |
|---------|------|
| `Array.flat()` | 数组扁平化，处理多维数组 |
| `Array.flatMap()` | 先映射再扁平化数组 |
| `String.trimStart()` | 去除字符串开头空格 |
| `String.trimEnd()` | 去除字符串结尾空格 |
| `Object.fromEntries()` | 将键值对数组转为对象 |

---

## ES11 (2020)
| 方法/语法 | 作用 |
|---------|------|
| 可选链 `?.` | 安全读取深层对象属性，防止报错 |
| 空值合并 `??` | 仅 `null/undefined` 时使用默认值 |
| `Promise.allSettled()` | 等待所有 Promise 完成，返回所有结果 |

---

## ES12 (2021)
| 方法/语法 | 作用 |
|---------|------|
| `String.replaceAll()` | 全局替换字符串，无需正则 |
| 逻辑赋值 `||= &&= ??=` | 逻辑运算 + 赋值合并简写 |

---

## ES13 (2022)
| 方法/语法 | 作用 |
|---------|------|
| `Array.at()` | 支持正负索引，快速获取数组指定项（取最后一项神器） |
| 类私有属性 `#` | 定义类内部私有属性，外部无法访问 |
| 顶层 `await` | 模块顶层直接使用 await，无需 async 函数 |
