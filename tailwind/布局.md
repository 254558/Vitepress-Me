# 布局

## 1. 最常用：Flex 弹性布局（导航、卡片、横向排列）
### 基础横向布局
<div class="flex gap-4 items-center">
  <div class="p-4 bg-blue-100">左侧</div>
  <div class="p-4 bg-green-100">中间</div>
  <div class="p-4 bg-yellow-100">右侧</div>
</div>

```html
<div class="flex gap-4 items-center">
  <div class="p-4 bg-blue-100">左侧</div>
  <div class="p-4 bg-green-100">中间</div>
  <div class="p-4 bg-yellow-100">右侧</div>
</div>
```

### 常用 Flex 类速查
- `flex`：开启 flex
- `flex-col`：垂直排列
- `justify-center`：水平居中
- `items-center`：垂直居中
- `justify-between`：两端对齐
- `gap-4`：元素间距（1=4px）
- `flex-1`：自动占满剩余空间

### 经典导航栏
<nav class="flex justify-between items-center p-4 bg-white shadow">
  <div class="text-lg font-bold">Logo</div>
  <ul class="flex gap-6">
    <li>首页</li>
    <li>产品</li>
    <li>关于</li>
  </ul>
</nav>

## 2. 响应式网格布局（Grid，卡片列表、多列布局）
### 4 列网格 + 响应式

<div class="grid grid-cols-1 sm:grid-cols-2 md:grid-cols-3 lg:grid-cols-4 gap-6">
  <div class="p-6 bg-white rounded shadow">卡片 1</div>
  <div class="p-6 bg-white rounded shadow">卡片 2</div>
  <div class="p-6 bg-white rounded shadow">卡片 3</div>
  <div class="p-6 bg-white rounded shadow">卡片 4</div>
</div>

```html
<div class="grid grid-cols-1 sm:grid-cols-2 md:grid-cols-3 lg:grid-cols-4 gap-6">
  <div class="p-6 bg-white rounded shadow">卡片 1</div>
  <div class="p-6 bg-white rounded shadow">卡片 2</div>
  <div class="p-6 bg-white rounded shadow">卡片 3</div>
  <div class="p-6 bg-white rounded shadow">卡片 4</div>
</div>
```

### Grid 核心类
- `grid`：开启网格
- `grid-cols-3`：3 列
- `gap-6`：间距
- 响应式：`sm:` `md:` `lg:` 自动适配手机/平板/电脑


## 3. 经典页面布局（Header + 侧边栏 + 主内容 + Footer）

<div class="h-[500px] flex flex-col">
  <!-- 头部 -->
  <header class="p-4 bg-gray-800 text-white">Header</header>

  <!-- 主体：侧边栏 + 内容 -->
  <div class="flex flex-1">
    <aside class="w-32 p-4 bg-gray-500">侧边栏</aside>
    <main class="flex-1 p-6">主内容区域</main>
  </div>

  <!-- 底部 -->
  <footer class="p-4 bg-gray-800 text-white">Footer</footer>
</div>

```html
<div class="min-h-screen flex flex-col">
  <!-- 头部 -->
  <header class="p-4 bg-gray-800 text-white">Header</header>

  <!-- 主体：侧边栏 + 内容 -->
  <div class="flex flex-1">
    <aside class="w-64 p-4 bg-gray-100">侧边栏</aside>
    <main class="flex-1 p-6">主内容区域</main>
  </div>

  <!-- 底部 -->
  <footer class="p-4 bg-gray-800 text-white">Footer</footer>
</div>
```

## 4. 居中布局（最常用）
### 水平 + 垂直居中

<div class="flex justify-center items-center h-screen">
  <div class="p-8 bg-white shadow rounded">我在正中间</div>
</div>

```html
<div class="flex justify-center items-center h-screen">
  <div class="p-8 bg-white shadow rounded">我在正中间</div>
</div>
```

### 页面内容最大宽度居中（网站通用）
```html
<div class="max-w-7xl mx-auto px-4">
  内容永远居中，左右有留白，超大屏不会变形
</div>
```

## 5. 响应式移动端适配（必备）
Tailwind 自带断点，不用写媒体查询：
- `sm:` ≥640px
- `md:` ≥768px
- `lg:` ≥1024px
- `xl:` ≥1280px

示例：手机单列、桌面双列

<div class="flex flex-col md:flex-row gap-6">
  <div class="flex-1 p-4 bg-blue-50">手机占满，桌面一半</div>
  <div class="flex-1 p-4 bg-green-50">手机占满，桌面一半</div>
</div>

# 快速记忆口诀
- **横向排列**：`flex`
- **垂直排列**：`flex-col`
- **间距**：`gap-*`
- **居中**：`justify-center items-center`
- **网格**：`grid grid-cols-*`
- **响应式**：加 `md:` `lg:` 前缀
- **页面居中**：`max-w-7xl mx-auto`

---

### 总结
1. Tailwind 布局**不用写 CSS**，全靠工具类实现
2. 日常 90% 布局用 **Flex + Grid** 就能搞定
3. 响应式直接加断点前缀，非常简单
